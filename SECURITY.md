# Security Policy

## Supported Versions

Currently, only the latest `main` branch version of Remind Karo is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security is a high priority for us. If you discover a security vulnerability within Remind Karo, please **DO NOT** open a public issue.

Instead, please send an email to **kartikpant.kp69@gmail.com**.

We will try to acknowledge your report within 48 hours, and we'll keep you updated on our progress towards a fix. Once the issue is resolved, we will publish a security advisory and credit you for the discovery if you wish.

Thank you for helping to keep our project safe!
